###### *Stable Version 1.0.0*
# Morse Over Sock

***Experimental Project, Not Suitable for Daily Usage***
## Installing
First, install latest .whl release from releases page
```
$ cd ~/Downloads
$ python3 -m pip install ./morse-0.0.0-py3-none-any.whl
```
## Example Usage

```
$ dvtm -m +
> + c
> + 1
$ python3 -m morse-over-sock
> + 2
$ python3 -m morse-over-sock
```
